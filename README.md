# 🎸 We Will Debug You - Algoritmo em Python

Um algoritmo simples e criativo em Python que toca a música "We Will Debug You" (paródia de "We Will Rock You") com ritmo, versos e refrões!

## 📋 Descrição

Este projeto é uma implementação educativa que demonstra conceitos fundamentais de Python:
- **Listas (Arrays)** para armazenar os versos
- **Funções** para encapsular lógica reutilizável
- **Loops com enumerate()** para iteração com controle de índice
- **Condicionais (If)** com operador módulo (%)
- **Formatação de strings** com f-strings
- **Controle de tempo** com `time.sleep()`

## 🎯 Funcionalidades

✅ Toca o ritmo característico (TUM TUM PÁ!)  
✅ Alterna entre refrão (MAIÚSCULO) e versos (minúsculo)  
✅ Usa emojis para melhorar a visualização  
✅ Implementa delay de tempo para sincronização  

## 💻 Como Usar

1. **Clone ou copie o arquivo** `index.py`
2. **Execute no terminal:**
   ```bash
   python index.py
   ```
3. **Aproveite a música! 🎵**

## 🔧 Conceitos Abordados

| Conceito | Descrição |
|----------|-----------|
| **Lista** | Armazena os versos da música |
| **Função** | `tocar_ritmo()` encapsula a lógica do ritmo |
| **Loop** | `for` com `enumerate()` para iterar com índice |
| **Condicional** | `if/else` com operador `%` (módulo) |
| **String** | f-strings e métodos `.upper()` / `.lower()` |
| **Time** | `time.sleep()` para sincronizar o ritmo |
